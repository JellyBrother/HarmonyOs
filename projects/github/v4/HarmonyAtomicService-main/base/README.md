### 基础库

| 模块名称                | 描述                  | 备注                                                                       |
|---------------------|---------------------|--------------------------------------------------------------------------|
| **fast_util**       | 工具类                 | FastLog（日志打印）<br/>EventBus（消息通知）                                         |
| **fast_ui**         | 提供 UI 封装、样式、toast 等 | CommonText（统一文本）<br/>CommonButton（统一按键）<br/> FastToast <br/> FastLoading |
| **global_constant** | 记录全局常量配置            | PagePathConstants(路由路径管理)                                                |
